# Whisperwood Villa

1. Update `db_config.py` with your PostgreSQL settings.
2. Install dependencies with `pip install -r requirements.txt`.
3. Run `python main.py`.

## App flow
- Splash screen
- Login window
- Dashboard after successful login

git init
git add .
git commit -m "Initial Whisperwood Villa app"
